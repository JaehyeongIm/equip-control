### TC-01: TEMP_HIGH 알람 전이 및 응답 시간

| 항목 | 내용 |
|-----|------|
| **테스트 ID** | TC-01 |
| **목적** | SHT31 온도 > 65°C 시 3초 이내 ALARM 전이 및 자동 대응 확인 |
| **관련 요구사항** | EFS-001 §7.2, AIM-001 ALID=1 |
| **합격 기준** | 65°C 초과 감지 후 3초 이내 PWM=0%, 팬 ON, 부저 ON, S5F1 보고 |

**사전 조건:**
- 장비 IDLE 상태
- EC 및 Host 정상 연결 (Online: YES)
- 챔버 도어 닫힘

**절차:**

| 단계 | 조작 | 확인 항목 |
|-----|------|---------|
| 1 | Host: `START` 명령 입력 | State = HEATING, PWM 동작 확인 |
| 2 | PWM Duty 최대(100%) 설정 또는 밀폐 챔버 가열 진행 | Host 온도 상승 확인 |
| 3 | SHT31 온도 65°C 초과 시점 기록 | 시각 T0 기록 |
| 4 | Host 화면 상태 전이 확인 | ALARM 전이 시각 T1 기록 |
| 5 | (T1 - T0) ≤ 3초 확인 | 응답 시간 측정 |
| 6 | 팬 동작 확인 | 팬 회전 소리/진동 |
| 7 | 부저 ON 확인 | 부저 경보음 |
| 8 | EC 로그: S5F1 ALID=1 전송 확인 | `[HSMS] S5F1 ALID=1 SET` |

**결과 기록:**

| 항목 | 기대값 | 측정값 | Pass/Fail |
|-----|--------|--------|----------|
| 응답 시간 (T1-T0) | ≤ 3s | | |
| State 전이 | → ALARM | | |
| PWM 상태 | 0% | | |
| 팬 동작 | ON | | |
| 부저 동작 | ON | | |
| S5F1 ALID=1 | 수신 | | |
