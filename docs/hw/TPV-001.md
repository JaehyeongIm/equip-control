## 1. 적용 범위

본 문서는 소형 챔버 온도제어 및 알람진단 시스템의 검증 테스트 계획 및 결과 기록 양식을 정의한다.

검증 대상:
- PID 온도 제어 정확도

- SENSOR_ERROR 감지 및 안전 정지
- COMM_ERROR 감지 및 FW 안전 모드
- 도어 열림 페일세이프
- EC RESET 복구 허용/거부 로직

### TC-02: TEMP_WARNING 감지 및 자동 대응

| 항목 | 내용 |
|-----|------|
| **테스트 ID** | TC-02 |
| **목적** | SP + warn_thr 초과 warn_dur 지속 시 WARNING 전이 및 부저 간헐 동작 확인 |
| **관련 요구사항** | EFS-001 §7.1, AIM-001 ALM-01 |
| **합격 기준** | 조건 충족 후 3초 이내 STATE = WARNING, 부저 간헐 ON |

**사전 조건:**
- HEATING 상태
- SP = 30°C, warn_thr = 3°C, warn_dur = 5s

**절차:**

| 단계 | 조작 | 확인 항목 |
|-----|------|---------|
| 1 | 챔버 밀폐 또는 PWM Duty 강제 상승으로 온도 > 33°C 유도 | 온도 상승 확인 |
| 2 | 33°C 초과 시점 기록 (T0) | |
| 3 | 5초 경과 후 EC 화면 확인 | STATE = WARNING 전이 시각 T1 |
| 4 | (T1 - T0) ≤ 3s 확인 (warn_dur 5s + 판정 지연) | |
| 5 | 부저 간헐 동작 확인 | 0.5s ON / 0.5s OFF |
| 6 | EC: `EVENT:WARN,ALM-01` 수신 확인 | |
| 7 | 온도 33°C 미만 복귀 시 자동 해제 확인 | `EVENT:CLEAR,ALM-01` 수신 |

**결과 기록:**

| 항목 | 기대값 | 측정값 | Pass/Fail |
|-----|--------|--------|----------|
| WARNING 전이 | T1-T0 ≤ 8s (warn_dur+판정) | | |
| 부저 간헐 동작 | 확인 | | |
| EVENT:WARN 수신 | 확인 | | |
| 자동 해제 | 온도 복귀 시 CLEAR | | |
