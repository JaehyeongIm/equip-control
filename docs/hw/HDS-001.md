## 3. MCU 선정

| 항목 | 내용 |
|-----|------|
| 보드 | STM32 NUCLEO-F446RE |
| MCU | STM32F446RE (ARM Cortex-M4, 180MHz, FPU) |
| 교체 경위 | STM32F411RE 사용 중 히터 회로 단락(스파크)으로 인한 보드 손상 → F446RE 교체 |
| 주요 페리페럴 | TIM3_CH3 (히터 PWM), USART2 (EC 통신), GPIO |
| Arduino 핀 매핑 | PB0=A3, PA0=A0, PA1=A1, PB5=D4, PB10=D6 |
| EC 통신 경로 | ST-LINK USB VCP → USART2 (PA2 TX / PA3 RX) |

## 4. 온도 센서: DS18B20

| 항목 | 내용 |
|-----|------|
| 제품 | DS18B20 (Maxim Integrated) |
| 인터페이스 | OneWire (단선 통신, 1 GPIO) |
| 정확도 | ±0.5°C (−10°C ~ +85°C) |
| 해상도 | 12-bit (기본), 750ms 변환 시간 |
| 연결 핀 | PB5 / D4 |
| 풀업 저항 | 4.7kΩ (DQ ← 3.3V) 필수 |
| 전원 | 외부 3.3V (기생전원 미사용) |
| DHT22 대비 선정 이유 | DHT22는 최소 2초 측정 간격 제한. DS18B20은 750ms 주기로 PID 제어에 적합. 디지털 출력으로 노이즈에 강함 |

**배선:**
```
STM32 3.3V ──┬── 4.7kΩ ──┬── DS18B20 DQ (PB5/D4)
             │            │
STM32 3.3V ──┘            └── DS18B20 VDD
STM32 GND ──────────────────── DS18B20 GND
```

## 5. 히터 제어 회로

### 5.1 히터 사양

| 항목 | 내용 |
|-----|------|
| 종류 | 카프톤 필름 히터 |
| 전압 | 24V DC |
| 저항 | ~22Ω |
| 최대 전력 | 약 26W |
| 동작 방식 | N-ch MOSFET PWM 제어 (1kHz) |

### 5.2 MOSFET 모듈 선정

| 항목 | 내용 |
|-----|------|
| 구성 | N-ch MOSFET 모듈 (납땜 불필요, 터미널 블록 방식) |
| 요구 사양 | VGS(th) ≤ 3.3V (STM32 GPIO 직접 구동), VDS ≥ 30V, ID ≥ 2A |
| 게이트 구동 | PB0(A3) 3.3V PWM → 게이트 직렬 저항 220Ω |
| 게이트 풀다운 | 10kΩ (Gate-GND, MCU 리셋/High-Z 시 OFF 보장) |
| PWM 주파수 | 1kHz (TIM3_CH3) |
| PWM Duty | 0 ~ 100% (Compare 0 ~ 1000) |

> **중요:** VGS(th) > 4V 제품은 3.3V GPIO 직접 구동 불가. 로직 레벨 구동 MOSFET 선택 필수.

### 5.3 PWM 타이머 설정

| 항목 | 값 |
|-----|-----|
| 타이머 | TIM3 |
| 채널 | CH3 (PB0, AF2) |
| APB1 클럭 | 84MHz (STM32F446RE: PLLCLK=168MHz, APB1 분주 /2) |
| Prescaler | 83 → 타이머 클럭 = 1MHz |
| Period (ARR) | 999 → 주파수 = 1MHz / 1000 = 1kHz |
| Compare 계산 | Duty% × 10 (예: 20% → Compare = 200) |

### 5.4 게이트 구동 회로 설계 근거

**직렬 저항 220Ω:**
- 게이트 충방전 전류 제한 → 스위칭 시 EMI 감소
- STM32 GPIO 출력 전류 보호 (최대 25mA 소스)

**풀다운 저항 10kΩ:**
- MCU 리셋 또는 GPIO High-Z 구간에서 MOSFET OFF 상태 보장
- 의도치 않은 히터 가동 방지 (안전 설계)

## 6. 냉각 팬 회로

| 항목 | 내용 |
|-----|------|
| 팬 사양 | 5V DC 팬 |
| 전원 경로 | WANPTEK 24V → DC-DC 컨버터 (24V→5V) → 릴레이 모듈 → 팬 |
| 제어 방식 | 릴레이 모듈 ON/OFF (PA0/A0, GPIO Output) |
| DC-DC 요구 조건 | 입력 전압 > 출력 전압 (24V 입력 시 5V 출력 가능. 5V 입력으로는 동작 불가) |
| 동작 조건 | ALARM 상태 또는 온도 ≥ SP - 1°C 시 ON |

## 7. 부저 회로

| 항목 | 내용 |
|-----|------|
| 종류 | 액티브 부저 (내부 발진 회로 내장, 외부 신호 불필요) |
| 전원 | 3.3V |
| 제어 핀 | PB10 / D6 (GPIO Output) |
| 동작 | HIGH → 부저 ON |
| WARNING 패턴 | 0.5s ON / 0.5s OFF 간헐 (FW 소프트웨어 구현) |
| ALARM 패턴 | 연속 ON |

> **CubeMX 설정 필요:** PB10을 GPIO_Output으로 추가 설정. 현재 미구성 상태.

## 8. 도어 스위치 회로

| 항목 | 내용 |
|-----|------|
| 스위치 종류 | NO (Normal Open) 리미트 스위치 |
| 연결 핀 | PA1 / A1 (GPIO Input, 내부 Pull-Up) |
| 도어 닫힘 | 스위치 눌림 → PA1 = LOW |
| 도어 열림 | 스위치 해제 또는 배선 단선 → PA1 = HIGH |
| 페일세이프 근거 | Pull-Up + NO 조합: 배선 단선 시 HIGH = 도어 열림 감지 → 히터 OFF (안전). NC 사용 시 단선 = LOW = 닫힘 오인 → 위험 |

## 9. 전원 아키텍처

```
WANPTEK 24V PSU (1.5A 전류 제한)
       │
       ├── 24V ──── MOSFET 모듈 ──── 카프톤 히터 (~1.08A max)
       │
       ├── 24V ──── DC-DC (24V→5V) ──── 릴레이 모듈 ──── 5V 팬
       │
USB ───┤── STM32 LDO (3.3V) ──── DS18B20
       │                    └─── 부저
       │                    └─── 릴레이 IN 신호
       │                    └─── GPIO
       └── GND (공통)
```

| 레일 | 전압 | 공급처 | 주요 부하 |
|------|------|--------|---------|
| 24V_HEATER | 24V | WANPTEK PSU | MOSFET → 카프톤 히터 |
| 5V_FAN | 5V | DC-DC 컨버터 | 릴레이 → 냉각 팬 |
| 3V3 | 3.3V | STM32 LDO (CN6-4) | DS18B20, 부저 |
| GND | 0V | CN6-6/7 | 전체 공통 접지 |

**PSU 전류 제한 1.5A 설정 이유:** MOSFET 단락 등 회로 사고 시 과전류 자동 차단. 카프톤 히터 정격 전류 ~1.08A 이므로 정상 운전에서 충분한 여유.
