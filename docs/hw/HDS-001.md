## 4. 히터 PWM 제어 회로 (핵심 설계)

### 4.1 설계 개요

히터 전력을 ON/OFF 릴레이가 아닌 **MOSFET PWM**으로 제어한다.  
이를 통해 설정 온도에 따라 히터 출력(0~100%)을 연속 조절하여 과온 위험을 줄이고  
온도 제어 정밀도를 개선한다.

| 항목 | 릴레이 방식 (구) | MOSFET PWM 방식 (신) |
|-----|--------------|-------------------|
| 제어 | ON/OFF 이진 | Duty 0~100% 연속 |
| 수명 | 기계적 마모 (~10만 회) | 반도체, 반영구 |
| 응답 속도 | ~10ms | ~1μs |
| PWM 주파수 | 불가 | 1kHz |
| 3.3V 구동 | 코일 구동 필요 | 직접 구동 가능 (IRLZ44NPBF) |

### 4.2 소자 선정: IRLZ44NPBF

| 파라미터 | 값 | 비고 |
|---------|-----|------|
| 형식 | N-ch Enhancement MOSFET | |
| VDS(max) | 55V | 24V 회로 대비 충분한 여유 |
| ID(max) | 47A | 히터 전류 1.08A 대비 충분 |
| VGS(th) | 1.0V (min) ~ 2.0V (max) | **3.3V GPIO 직접 구동 가능** |
| RDS(on) | 22mΩ @ VGS=10V | 전압 강하 무시 가능 |
| 패키지 | TO-220 | 방열판 부착 가능 |
| 비교 기준 | IRF520: VGS(th)≤4V → 3.3V 구동 부적합 | IRLZ44NPBF 선택 이유 |

### 4.3 히터 PWM 회로도

```
  24V DC ─────────── ST-22 ──────────── INA219 VIN+ ─── [0.1Ω Shunt] ─── INA219 VIN-
                    (70°C 차단)                                                  │
                                                                                │
                                                              카프톤 히터 (22Ω, 26W)
                                                                                │
                                                    ┌───────────────────────────┘
                                                    │  IRLZ44NPBF
                                                    │  ├─ Drain ◄──── 히터 (-) 단자
                                                    │  ├─ Source ──── GND
                                                    │  └─ Gate ◄──── 100Ω ◄──── PB0 (3.3V PWM)
                                                    │                 │
                                                    │               10kΩ
                                                    │                 │
                                                    └─────────────── GND

  GND ──────────────────────────────────────────────────────────────────────── GND
```

### 4.4 게이트 구동 회로 설명

**직렬 저항 (100Ω):**
- 게이트 충방전 전류 제한 → 스위칭 시 EMI 감소
- GPIO 출력 전류 보호 (STM32 GPIO 최대 소스 전류: 25mA)
- 게이트 전하 Qg ≈ 48nC, 100Ω에서 스위칭 시간 ≈ 5μs (1kHz PWM에서 문제 없음)

**풀다운 저항 (10kΩ):**
- MCU 리셋 또는 GPIO High-Z 상태에서 MOSFET이 불확정 상태가 되는 것을 방지
- MOSFET OFF 상태 보장 → 의도치 않은 히터 가동 방지 (안전)

**동작점 확인:**
- VGS = 3.3V (HIGH): 3.3V > VGS(th) max 2.0V → 완전 도통 (Fully ON)
- VGS = 0V (LOW): 0V < VGS(th) min 1.0V → 완전 차단 (Fully OFF)

### 4.5 PWM 파라미터

| 파라미터 | 값 | 근거 |
|---------|-----|------|
| TIM 채널 | TIM3_CH3 (AF2) | PB0 대응 채널 |
| PWM 주파수 | 1kHz (주기 1ms) | 히터 열 응답 시정수(수십 초) 대비 충분히 빠름 |
| Duty 범위 | 0~100% | 0%=완전 OFF, 100%=최대 출력 26W |
| ARR 값 | 83 (84MHz / 1kHz 분주) | TIM3 Prescaler 설정에 따라 조정 |
| 제어 범위 | 0~26W (0~1.08A) | 히터 정격 24V/26W 기준 |

## 5. 이중 인터락 구조

과온 장애에 대해 소프트웨어와 하드웨어 두 계층의 인터락을 구성한다.

```
                      챔버 온도 상승
                            │
         ┌──────────────────┼──────────────────┐
         │                  │                  │
      65°C 초과          68°C 초과          70°C 도달
         │                  │                  │
    ┌────▼─────┐       ┌────▼──────┐      ┌────▼──────┐
    │ SW ALARM  │       │ SW        │      │ HW        │
    │ EC 감지   │       │ INTERLOCK │      │ INTERLOCK │
    │ PWM 감소  │       │ PWM=0%    │      │ ST-22     │
    │ 팬 ON     │       │ S5F1 보고 │      │ 물리 차단  │
    │ 부저 ON   │       │           │      │           │
    └───────────┘       └───────────┘      └────┬──────┘
                                                │
                                     INA219 전류 소실 감지
                                     (0mA → EC에서 HW 동작 확인)
```

| 계층 | 동작 조건 | 구현 | 복구 방법 |
|------|---------|------|---------|
| SW ALARM | SHT31 온도 > 65°C | EC 소프트웨어, PWM Duty 0% | 온도 < 50°C 후 RESET |
| SW INTERLOCK | SHT31 온도 > 68°C | EC 소프트웨어, PWM Duty 0% | 온도 < 50°C 후 RESET |
| HW INTERLOCK | 챔버 70°C 도달 | ST-22 자동 개방 | 냉각 후 자동 복귀 (운영자 확인 필요) |

## 6. 온도/전류 센서 인터페이스

### 6.1 SHT31 (Sensirion) — 온습도 센서

| 항목 | 사양 |
|-----|------|
| 인터페이스 | I2C (100kHz, 표준 모드) |
| I2C 주소 | 0x44 (ADDR 핀 → GND) |
| 온도 정확도 | ±0.3°C (typ), ±0.7°C (max) |
| 습도 정확도 | ±2%RH (typ) |
| 동작 전압 | 2.4V ~ 5.5V (3.3V 공급) |
| 측정 주기 | 1초 (FW TaskSensor 기준) |
| 역할 | 과온 감지 (ALARM 65°C, INTERLOCK 68°C), 온도 상승 지연 감지 |

### 6.2 INA219 (TI) — 전류/전압 센서

| 항목 | 사양 |
|-----|------|
| 인터페이스 | I2C (100kHz, 표준 모드) |
| I2C 주소 | 0x40 (A0, A1 핀 → GND) |
| 측정 범위 | 전압: 0~26V, 전류: 0~3.2A |
| 분해능 | 12bit |
| Shunt 저항 | 0.1Ω (모듈 내장) |
| 동작 전압 | 3.0V ~ 5.5V (3.3V 공급) |
| 역할 | 히터 전류 감시, ST-22 동작 감지 |

**히터 정상 전류 범위:**
- 히터 저항 ≈ 22Ω (24V / 26W)
- 정격 전류 = 24V / 22Ω ≈ **1083mA**
- 정상 범위: 800mA ~ 1300mA (저항 허용 오차 ±20%, 전압 변동 고려)
- 이상 판정: < 50mA (회로 차단) 또는 < 300mA (30% 미만, 부분 단선)

### 6.3 I2C 버스 공유 설계

SHT31과 INA219는 동일한 I2C1 버스(PB8/PB9)를 공유한다.

```
  3.3V ─┬── 4.7kΩ ──┬─── PB8 (SCL)
        │           ├─── SHT31 SCL
        │           └─── INA219 SCL
        │
        └── 4.7kΩ ──┬─── PB9 (SDA)
                    ├─── SHT31 SDA
                    └─── INA219 SDA
```

- I2C 주소 충돌 없음: SHT31(0x44) ≠ INA219(0x40)
- 풀업 저항 4.7kΩ: 100kHz 표준 모드 기준 충분한 풀업 강도
- 배선 길이: 브레드보드 내 30cm 이하 → 추가 커패시터 불필요

## 7. 팬 제어 회로

히터 회로와 동일하게 **IRLZ44NPBF**로 구성한다. IRF520 모듈은 사용하지 않는다.

| 항목 | 내용 |
|-----|------|
| 소자 | IRLZ44NPBF (히터 회로와 동일) |
| VGS(th) | 1.0V (min) ~ 2.0V (max) |
| 구동 | STM32 GPIO HIGH = 3.3V → 완전 도통 보장 |
| Gate 저항 | 100Ω 직렬 + 10kΩ 풀다운 (히터 회로와 동일 구성) |
| 전원 | DC/DC 컨버터 5V 출력 |

```
DC/DC OUT+(5V) ──── 팬(+)
팬(-) ──── IRLZ44NPBF Drain
           Source ──── GND
           Gate ◄── 100Ω ◄── PB4 (TIM3_CH1)
                       ↕ 10kΩ
                      GND
```
