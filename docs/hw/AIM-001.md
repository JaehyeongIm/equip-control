## 1. 적용 범위

본 문서는 소형 챔버 온도제어 및 알람진단 시스템의 알람 목록, 발생 조건, 자동 대응 동작, 복구 방법을 정의한다.

대상 알람 시나리오:
- 과온 WARNING / ALARM (DS18B20 기준)
- 센서 오류 (SENSOR_ERROR)
- 통신 단선 (COMM_ERROR)
- 도어 열림 (DOOR_OPEN)

## 3. 시스템 상태 정의

| 상태 | 설명 | 히터 | 팬 | 부저 |
|------|------|------|-----|------|
| IDLE | 대기. 가열 정지 | OFF | OFF | OFF |
| HEATING | PID 제어 중. SP 추종 | PWM | 조건부 ON | OFF |

## 4. 알람 매트릭스

| ALID | 알람명 | 우선순위 | 발생 조건 | FW 자동 대응 | 복구 방법 |
|------|--------|---------|---------|------------|---------|
| ALM-01 | TEMP_WARNING | WARNING | DS18B20 온도 > SP + warn_thr (기본 3°C), warn_dur (기본 5s) 이상 지속 | 부저 간헐 ON, EC WARNING 보고, PID 유지 | 온도 ≤ SP + warn_thr → 자동 해제 |
| ALM-02 | TEMP_ALARM | ALARM | DS18B20 온도 > SP + alarm_thr (기본 5°C), alarm_dur (기본 10s) 이상 지속 | 히터 PWM=0%, 팬 ON, 부저 연속 ON, EC ALARM 보고 | 온도 복귀 후 EC `RESET` 명령 (수동) |
| ALM-03 | SENSOR_ERROR | ALARM | DS18B20 읽기 3회 연속 실패 (CRC 오류 또는 응답 없음) | 히터 PWM=0%, 팬 OFF, 부저 ON, EC ERROR 보고 | 센서 복구 후 EC `RESET` 명령 (수동) |
| ALM-04 | COMM_ERROR | ERROR | EC ↔ FW UART Heartbeat 10초 미수신 | 히터 PWM=0%, 팬 OFF (FW 자체 안전 모드) | EC 재연결 → Heartbeat 재개 → 자동 해제 |
| ALM-05 | DOOR_OPEN | WARNING | 도어 스위치 PA1 HIGH (도어 열림 또는 배선 단선) | 히터 PWM=0%, EC WARN 보고 | 도어 닫힘 (PA1 LOW) → 자동 해제 |

## 5. 알람 상세

### ALM-01: TEMP_WARNING

| 항목 | 내용 |
|-----|------|
| 발생 조건 | DS18B20 온도 > SP + warn_thr, warn_dur 초 이상 지속 |
| 기본 임계값 | warn_thr = 3°C, warn_dur = 5s (EC에서 설정 가능) |
| 예시 | SP=30°C → 33°C 초과 5초 지속 시 발생 |
| FW 동작 | 부저 간헐 ON (0.5s ON / 0.5s OFF), PID 제어 유지 |
| EC 동작 | WARNING 패널 표시, 이벤트 로그 기록 |
| 자동 해제 | 온도 ≤ SP + warn_thr 시 자동 해제, EC CLEAR 보고 |
| 비고 | TEMP_ALARM 조건 지속 시 ALM-02로 에스컬레이션 |

### ALM-02: TEMP_ALARM

| 항목 | 내용 |
|-----|------|
| 발생 조건 | DS18B20 온도 > SP + alarm_thr, alarm_dur 초 이상 지속 |
| 기본 임계값 | alarm_thr = 5°C, alarm_dur = 10s (EC에서 설정 가능) |
| 예시 | SP=30°C → 35°C 초과 10초 지속 시 발생 |
| FW 동작 | 히터 PWM=0%, 팬 ON, 부저 연속 ON, STATE → ALARM |
| EC 동작 | ALARM 패널 표시, 진단 가이드 표시, 이벤트 로그 기록 |
| 복구 조건 | 온도 ≤ SP - 2°C (기본값) |
| RESET 거부 조건 | 복구 온도 미충족 시 NACK 반환 및 사유 표시 |

### ALM-03: SENSOR_ERROR

| 항목 | 내용 |
|-----|------|
| 발생 조건 | DS18B20 읽기 3회 연속 실패 (CRC 불일치 또는 응답 없음) |
| FW 동작 | 히터 PWM=0%, 팬 OFF, 부저 ON, STATE → ALARM |
| EC 동작 | SENSOR_ERROR 패널 표시, 센서 점검 안내 |
| 복구 방법 | 센서 배선 점검 및 복구 후 EC `RESET` 명령 |

### ALM-04: COMM_ERROR

| 항목 | 내용 |
|-----|------|
| 발생 조건 | FW가 EC로부터 Heartbeat를 10초 동안 미수신 |
| FW 동작 | 히터 PWM=0%, 팬 OFF (FW 단독 안전 모드 진입) |
| EC 동작 | 연결 단선 표시, COMM_ERROR 패널 표시 |
| 자동 해제 | EC 재연결 후 Heartbeat 재개 → FW 자동 해제 |

### ALM-05: DOOR_OPEN

| 항목 | 내용 |
|-----|------|
| 발생 조건 | 도어 스위치 PA1 = HIGH (도어 열림 또는 배선 단선) |
| FW 동작 | 히터 PWM=0%, EC WARN 보고 |
| 설계 의도 | 페일세이프: 배선 단선 시 히터 정지 (안전) |
| 자동 해제 | PA1 = LOW (도어 닫힘) 감지 시 자동 해제 |

## 6. 상태 전이 요약

```
               START
IDLE ────────────────────► HEATING
 ▲                              │
 │ RESET (조건 충족)             │ ALM-01 (TEMP_WARNING)
 │                              ▼
 │                          WARNING
 │                              │ 온도 복귀
 │                              ├──────────────────── HEATING
 │                              │
 │              ALM-02 / ALM-03 │
 │         ┌────────────────────┘
 │         ▼
ALARM ◄──────────────────── HEATING (ALM-02 / ALM-03 직접)
 │
 └── RESET (조건 충족) ──────────────────────► IDLE

ALM-04 / ALM-05: 인라인 조건, 히터 즉시 OFF (상태 전이 없음)
```

## 7. 복구 조건 매트릭스

| ALID | 복구 유형 | 조건 | EC 명령 필요 |
|------|---------|------|------------|
| ALM-01 | 자동 | 온도 ≤ SP + warn_thr | 불필요 |
| ALM-02 | 수동 | 온도 ≤ SP - 2°C | `RESET` 필요 |
| ALM-03 | 수동 | 센서 정상 복귀 확인 | `RESET` 필요 |
| ALM-04 | 자동 | Heartbeat 재개 | 불필요 |
| ALM-05 | 자동 | PA1 = LOW | 불필요 |
