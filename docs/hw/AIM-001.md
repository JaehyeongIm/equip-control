## 4. 알람 매트릭스

| ALID | 알람명 | 우선순위 | 발생 조건 | FW 자동 대응 | 복구 방법 |
|------|--------|---------|---------|------------|---------|
| ALM-01 | TEMP_WARNING | WARNING | 온도 > SP + warn_thr (기본 3°C), warn_dur (기본 5s) 이상 지속 | 부저 간헐 ON, EC WARNING 보고, PID 유지 | 온도 ≤ SP + warn_thr → 자동 해제 |
| ALM-02 | TEMP_ALARM | ALARM | 온도 > SP + alarm_thr (기본 5°C), alarm_dur (기본 10s) 이상 지속 | 히터 PWM=0%, 팬 ON, 부저 연속 ON, EC ALARM 보고 | 온도 복귀 후 EC `RESET` 명령 (수동) |
| ALM-03 | SENSOR_ERROR | ALARM | DHT22 읽기 3회 연속 실패 | 히터 PWM=0%, 팬 OFF, 부저 ON, EC ERROR 보고 | 센서 복구 후 EC `RESET` 명령 (수동) |

## 5. 알람 상세

### ALM-01: TEMP_WARNING

| 항목 | 내용 |
|-----|------|
| 발생 조건 | 온도 > SP + warn_thr, warn_dur 초 이상 지속 |
| 기본 임계값 | warn_thr = 3°C, warn_dur = 5s |
| 예시 | SP=30°C → 33°C 초과 5초 지속 시 발생 |
| FW 동작 | 부저 간헐 ON (0.5s ON / 0.5s OFF), PID 제어 유지 |
| EC 동작 | WARNING 패널 표시, 이벤트 로그 기록 |
| 자동 해제 | 온도 ≤ SP + warn_thr 시 자동 해제, EC CLEAR 보고 |
| 비고 | TEMP_ALARM 조건 지속 시 ALM-02로 에스컬레이션 |

### ALM-02: TEMP_ALARM

| 항목 | 내용 |
|-----|------|
| 발생 조건 | 온도 > SP + alarm_thr, alarm_dur 초 이상 지속 |
| 기본 임계값 | alarm_thr = 5°C, alarm_dur = 10s |
| 예시 | SP=30°C → 35°C 초과 10초 지속 시 발생 |
| FW 동작 | 히터 PWM=0%, 팬 ON, 부저 연속 ON, STATE → ALARM |
| EC 동작 | ALARM 패널 표시, 진단 가이드 표시, 이벤트 로그 기록 |
| 복구 조건 | 온도 ≤ SP - 2°C |
| RESET 거부 조건 | 복구 온도 미충족 시 NACK 반환 및 사유 표시 |

### ALM-03: SENSOR_ERROR

| 항목 | 내용 |
|-----|------|
| 발생 조건 | DHT22 읽기 3회 연속 실패 |
| FW 동작 | 히터 PWM=0%, 팬 OFF, 부저 ON, STATE → ALARM |
| EC 동작 | SENSOR_ERROR 패널 표시, 센서 점검 안내 |
| 복구 방법 | 센서 배선 점검 및 복구 후 EC `RESET` 명령 |

## 6. 상태 전이 요약

```
               START
IDLE ────────────────────► HEATING
 ▲                              │
 │ RESET (조건 충족)             │ ALM-01 (TEMP_WARNING)
 │                              ▼
 │                          WARNING
 │                              │ 온도 복귀
 │                              ├──────────────────── HEATING
 │                              │
 │              ALM-02 / ALM-03 │
 │         ┌────────────────────┘
 │         ▼
ALARM ◄──────────────────── HEATING (ALM-02 / ALM-03 직접)
 │
 └── RESET (조건 충족) ──────────────────────► IDLE

※ STOP 명령: HEATING / WARNING → IDLE
```

## 7. 복구 조건 매트릭스

| ALID | 복구 유형 | 조건 | EC 명령 필요 |
|------|---------|------|------------|
| ALM-01 | 자동 | 온도 ≤ SP + warn_thr | 불필요 |
| ALM-02 | 수동 | 온도 ≤ SP - 2°C | `RESET` 필요 |
| ALM-03 | 수동 | 센서 정상 복귀 확인 | `RESET` 필요 |
