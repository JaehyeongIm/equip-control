## 1. 적용 범위

본 문서는 가상 공정 챔버 장비(STM32 NUCLEO-F411RE 기반)의 전체 I/O 신호 목록과 배선 구성을 정의한다.  
과온(Over-Temperature) 및 온도 상승 지연(No-Heat) 시나리오 대응에 필요한 모든 신호가 포함된다.

## 5. 배선도 (Wiring Diagram)

### 5.1 히터 전력 회로 (12V, 핵심 회로)

```
┌─────────────────────────────────────────────────────────────────┐
│  24V DC (WANPTEK 파워 서플라이)                                 │
│  (+) 단자                                                       │
│      │                                                          │
│      │  12V ─────────────────────────────────┐                 │
│      │                                        │                 │
│      │                                   ┌────┴────┐           │
│      │                                   │  ST-22  │           │
│      │                              바이메탈│(70°C차단)│           │
│      │                                   └────┬────┘           │
│      │                                        │                 │
│      │                               ┌────────┴───────┐        │
│      │                               │   INA219       │        │
│      │                               │  VIN+ →[0.1Ω]→ VIN-   │
│      │                               │  SDA ──── PB9   │        │
│      │                               │  SCL ──── PB8   │        │
│      │                               │  VCC ──── 3.3V  │        │
│      │                               │  GND ──── GND   │        │
│      │                               └────────┬───────┘        │
│      │                                        │                 │
│      │                             ┌──────────┴──────────┐     │
│      │                             │   카프톤 히터        │     │
│      │                             │   12V / 10W / ~12Ω  │     │
│      │                             └──────────┬──────────┘     │
│      │                                        │                 │
│      │                            ┌───────────┴──────────────┐ │
│      │                            │   IRLZ44NPBF             │ │
│      │                            │   Drain ←─── 히터 (-)    │ │
│      │                            │   Source ──────────── GND │ │
│      │                            │   Gate ◄── 100Ω ◄── PB0  │ │
│      │                            │                ↕ 10kΩ     │ │
│      │                            │               GND         │ │
│      │                            └──────────────────────────┘ │
│      │                                                          │
│  DC Jack (-) ──────────────────────────────────────── GND      │
└─────────────────────────────────────────────────────────────────┘

PB0 (TIM3_CH3, AF2): PWM 출력 — 주파수 1kHz, Duty 0~100%
ST-22: 70°C 초과 시 자동 개방 → 히터 회로 물리 차단
INA219: 히터 전류/전압 실시간 측정 (ST-22 동작 감지용)
```

### 5.2 냉각 팬 회로 (5V)

```
USB VBUS (CN6-5) ── 5V ──────────────────────┐
                                              │
                              ┌───────────────┴────┐
                              │   냉각 팬           │
                              │   5V / 0.2A        │
                              └───────────────┬────┘
                                              │
                          ┌───────────────────┴──────────────┐
                          │   IRF520 MOSFET                  │
                          │   Drain ←─────────── 팬 (-)     │
                          │   Source ──────────────────── GND│
                          │   Gate ◄── 1kΩ ◄── PB4          │
                          └──────────────────────────────────┘

PB4 (TIM3_CH1, AF2): PWM 출력
주의: IRF520은 VGS(th) ≈ 4V로 3.3V 구동 부적합
      → 팬 회전 불안정 가능성 있음 (IRLZ44NPBF 교체 권장, HDS-001 참조)
```

### 5.3 센서 I2C 버스 (3.3V)

```
STM32 3.3V (CN6-4) ──┬── 4.7kΩ ──── PB8 (I2C1_SCL)

                                         │
                              ┌──────────┴──────────────────┐
                              │                             │
                    ┌─────────┴────────┐     ┌─────────────┴────┐
                    │  Sensirion SHT31  │     │  TI INA219       │

                    │  GND ──── GND     │     │  GND ──── GND    │
                    │  SCL ──── PB8     │     │  SCL ──── PB8    │
                    │  SDA ──── PB9     │     │  SDA ──── PB9    │

                    │  (주소: 0x44)     │     │  A1 ───── GND    │
                    └──────────────────┘     │  (주소: 0x40)    │

```

### 5.4 알람 및 상태 출력

```
STM32 3.3V ───── 액티브 부저 VCC
PB10      ───── 액티브 부저 S (Signal)
GND       ───── 액티브 부저 GND

PA5 (LD2) ───── 내장 녹색 LED (장비 상태 표시)
```

### 5.5 입력 신호 (도어, 버튼)

```
STM32 PA1 (내부 Pull-Up)
     │
     ├── HIGH (3.3V) : 스위치 눌림 → 도어 닫힘
     └── LOW  (GND)  : 스위치 개방 → 도어 열림 (알람)

     PA1 ──── 리미트 스위치 NC 단자
              리미트 스위치 COM ──── GND

STM32 PC13 (내부 Pull-Up, B1 내장 버튼)
     ├── HIGH : 미입력
     └── LOW  : 버튼 눌림 → Start/Stop 토글
```

### 5.6 UART 통신 (EC 연결)

```
STM32                    ST-LINK (USB VCP)          macOS EC
  PA2 (TX) ─────────────── TXD ──────────────────── /dev/tty.usbmodem*
  PA3 (RX) ─────────────── RXD
  GND      ─────────────── GND

  프레임: SOF(0xAA) | TYPE | SEQ | LEN(2B) | PAYLOAD | CRC16
  속도: 115200 bps, 8N1
```

## 6. 브레드보드 전원 레일 구성

```
MB-102 브레드보드 (상단 레일 / 하단 레일 독립)

상단 레일:  (+) ── 3.3V (STM32 CN6-4)
            (-) ── GND

하단 레일:  (+) ── 5V (USB VBUS, CN6-5)
            (-) ── GND

주의: 3.3V 레일과 5V 레일 연결 금지 (소자 손상)
```
