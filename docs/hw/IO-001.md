## 5. 배선도 (Wiring Diagram)

### 5.1 히터 제어 회로 (24V)

```
WANPTEK 24V PSU (+)
       │
       │ 24V ──────────────────────────────────────────┐
       │                                               │
       │                              ┌────────────────┴──────────┐
       │                              │   N-ch MOSFET 모듈        │
       │                              │   VIN+ ──── 카프톤 히터 (+) │
       │                              │   VIN- ──── 카프톤 히터 (-) │
       │                              │   PWM/Gate ◄── 220Ω ◄── PB0 │
       │                              │              ↕ 10kΩ       │
       │                              │             GND           │
       │                              └───────────────────────────┘
       │                                                          │
PSU (-) ───────────────────────────────────────────────────── GND

PB0 (TIM3_CH3, AF2): 1kHz PWM, Duty 0~100%
PSU 전류 제한: 1.5A (단락 보호)
```

### 5.2 냉각 팬 회로 (5V)

```
WANPTEK 24V ──── DC-DC 컨버터 (IN: 24V, OUT: 5V)
                        │
                 DC-DC OUT+ (5V)
                        │
              ┌──────────────────┐
              │   릴레이 모듈    │
              │   COM ── 5V      │
              │   NO  ──┐        │
              │   IN  ◄── PA0   │
              └──────────────────┘
                         │
                  ┌──────┴──────┐
                  │   5V DC 팬  │
                  └─────────────┘
                  팬(-) ──── GND

PA0 (A0): GPIO Output, HIGH=릴레이 ON=팬 ON
DC-DC 입력: 반드시 24V 이상 (5V 입력으로는 5V 출력 불가)
```

### 5.3 DHT22 온도 센서

```
STM32 3.3V (CN6-4) ──┬── 4.7kΩ ──┬── DHT22 DATA ──── PB5 (D4)
                     │            │
                     └────────────┴── DHT22 VDD
STM32 GND ────────────────────────── DHT22 GND

4.7kΩ 풀업 필수 (오픈 드레인 버스)
```

### 5.4 부저 및 상태 LED

```
STM32 3.3V ──── 액티브 부저 VCC
PB10 (D6)  ──── 액티브 부저 Signal
GND        ──── 액티브 부저 GND

PA5 (LD2)  ──── 내장 녹색 LED (장비 상태 표시)
```

### 5.5 UART 통신 (EC 연결)

```
STM32                  ST-LINK (USB VCP)         macOS EC
  PA2 (TX) ────────────── TXD ─────────────── /dev/tty.usbmodem*
  PA3 (RX) ────────────── RXD
  GND      ────────────── GND

프로토콜: ASCII 텍스트, \r\n 종단, 115200 bps 8N1
```
